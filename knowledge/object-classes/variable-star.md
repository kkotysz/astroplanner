# Variable Star Observing

Summary: Practical guidance for observing variable stars in AstroPlanner.
Tags: variable-star, star, practical-observing

## When this note applies

- The target is a variable star or similar stellar variable.
- The user asks how to observe FU Ori, T CrB, or another stellar variable.
- The user compares a variable star with other compact targets.

## Key heuristics

- Variable stars are usually compact stellar targets, so they tolerate moderate moonlight better than diffuse galaxies.
- For practical observing, prioritize a clean high-altitude window and low airmass over raw score alone.
- If the variable star is bright enough, it can be a very efficient target even when conditions are mediocre for faint extended objects.

## What to tell the user

- Focus on brightness, placement, and whether the target is a practical point source tonight.
- Do not invent specific variability behavior unless it is actually in the supplied metadata.
- If multiple stellar targets are available, compare them using altitude, best window, and brightness.

## Caveats

- The generic label `variable star` is broad and does not imply a specific observing strategy beyond point-source practicality.
- Catalog type alone does not tell you the current phase or scientific urgency.
