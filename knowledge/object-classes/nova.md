# Nova Observing

Summary: Practical guidance for classical and recurrent novae in AstroPlanner.
Tags: nova, transient, practical-observing

## When this note applies

- The target type or class is a nova.
- The user compares novae with supernovae or other transients.
- The user asks how to observe a nova tonight.

## Key heuristics

- Novae are transient point sources, so brightness, min airmass, and a clean observing window usually matter more than diffuse-object concerns.
- Do not treat novae and supernovae as interchangeable: both are transients, but the practical recommendation should still follow the actual brightness and geometry tonight.
- If a nova is bright and well placed, it is often a very efficient target for a small setup.

## What to tell the user

- Be explicit that this is a nova, not a supernova, if the distinction matters.
- Explain whether the target is currently practical because it is bright and compact, or marginal because it is faint or low.
- Compare it with other transients using brightness, min airmass, and best window.

## Caveats

- Catalog labels do not guarantee the current photometric state.
- A bright nova can still be a poor choice if the usable window is too short or the target is badly placed.
