# Supernova Observing

Summary: Practical guidance for choosing and observing supernovae in AstroPlanner.
Tags: supernova, sn, transient, practical-observing

## When this note applies

- The target type or class is a supernova.
- The user is choosing between multiple SN candidates.
- The user asks how to observe a supernova tonight.

## Key heuristics

- Prefer the supernova with the best combination of lower airmass, longer useful window, and reasonable brightness rather than chasing score alone.
- `Last Mag` from BHTOM matters: a brighter SN with a slightly worse score can still be the better practical target.
- If several SN are similar, prefer the one with the earlier closing window or the one currently best placed for a long uninterrupted run.

## What to tell the user

- Explain the tradeoff between brightness, min airmass, moon separation, and best window.
- If one SN is clearly brighter and still well placed, say that directly.
- If there is no clear winner, say which metric is deciding the tie.

## Caveats

- `Last Mag` may be stale if the transient is evolving quickly.
- A high score does not automatically mean the target is the easiest SN to capture on a small instrument.
