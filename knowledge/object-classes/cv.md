# Cataclysmic Variable Observing

Summary: Practical guidance for cataclysmic variables, dwarf novae, and related compact accreting systems.
Tags: cv, cataclysmic-variable, dwarf-nova, practical-observing

## When this note applies

- The target type or class is CV or cataclysmic variable.
- The user asks how to observe a compact eruptive variable tonight.
- The user is choosing between multiple CV-like targets.

## Key heuristics

- Treat CV targets as compact point sources: altitude, min airmass, and brightness usually matter more than moonlight penalties that dominate diffuse objects.
- If the object is in outburst or unusually bright, prioritize the cleanest available high-altitude window and keep the session continuous.
- For multiple CV candidates, prefer the brighter one unless another target has a much cleaner window or dramatically lower airmass.

## What to tell the user

- Frame the recommendation around brightness state, compactness, and clean geometry.
- Emphasize that a CV is usually more forgiving than a faint diffuse galaxy, but still benefits from low airmass.
- If the object is faint, say that longer total integration matters more than quick checks.

## Caveats

- Catalog brightness may not reflect the current outburst state.
- Some CV labels are broad and do not tell you whether the object is currently photometrically interesting.
