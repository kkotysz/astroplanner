# QSO and AGN Observing

Summary: Practical guidance for faint point-like AGN, quasars, and Seyfert targets.
Tags: qso, agn, seyfert, blazar, point-source, practical-observing

## When this note applies

- The target type contains QSO, quasar, AGN, Seyfert, or blazar.
- The user asks whether an AGN-like target is worth observing tonight.
- The user asks how to observe a faint extragalactic point source.

## Key heuristics

- For faint AGN and QSO targets, prioritize lower airmass and the best high-altitude segment over raw score alone.
- Moon separation usually matters more than for bright stars, especially if the target is faint and the sky is bright.
- Long continuous integrations are usually more useful than hopping between many AGN candidates.

## What to tell the user

- Focus on whether the target is realistically observable as a faint point source, not on galaxy morphology.
- Explain altitude, min airmass, and moon separation in practical terms.
- If the target is very faint, say that patience and longer total integration matter more than short looks.

## Caveats

- SIMBAD type is more reliable than a model guess about the target class.
- `Last Mag` from BHTOM may not represent the current brightness state for all AGN-like objects.
